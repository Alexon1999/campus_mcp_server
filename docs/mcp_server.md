Standardize way to give LLM models to interact with tools

c'est comme le protocole http pour les applications web, là c'est le protocole MCP server c'est pour les LLM (ai agent) qui peut communiquer avec ton programme

tu armes ton IA (LLM) avec tes outils à toi 

on va pouvoir lui donner des outils

- pour lire une base de connaissance (documentation)
- lui faire-faire des actions
